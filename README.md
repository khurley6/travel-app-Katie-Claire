# Travel App - Feature 6
Katie Hurley and Claire Huang

## Overview
This is a full-stack travel planning application that allows users to:
- Create and manage trips
- Add and organize itinerary items
- Authenticate with login and registration
- View trip destinations on an interactive map

## Features
- Component-based architecture using React
- Client-side routing with React Router
- Authentication with protected and public routes
- Asynchronous data handling with Parse backend
- Interactive map integration using Leaflet and OpenStreetMap
- Dynamic geocoding to convert destinations into coordinates

## How to Run

Install dependencies:
npm install

Start the development server:
npm run dev
